# Cursor Clone

A high-performance clone of [Cursor](https://www.cursor.com/) - the AI-native code editor built for extraordinary productivity.

## Overview

This project recreates the sleek, dark-themed landing page of Cursor. It features a sophisticated responsive layout, agentic feature showcases, industry-leading testimonials, and a modern developer-first aesthetic using Tailwind CSS.

## Project Structure

├── index.html          # Main HTML structure with Tailwind classes
├── src/style.css       # Tailwind directives and custom styles
├── public/             # Logos, feature images, and user avatars
└── README.md           # This file

## Features

- **Responsive Navigation:** Sticky header with mobile-optimized menu.
- **Agentic Showcases:** Interactive-style layouts for AI Agents and Tab Autocomplete.
- **Dynamic Testimonial Grid:** Responsive cards featuring leaders from NVIDIA, OpenAI, and Stripe.
- **Frontier Roadmap:** Sections for model selection and codebase indexing.
- **Changelog:** Minimalist version tracking and update feed.
- **Glassmorphism:** Modern UI depth using subtle borders and background opacities.

## Getting Started

1. Open `index.html` in your web browser to view the page.
2. Ensure Tailwind CSS is configured or linked in `src/style.css`.
3. Assets such as logos and screenshots are located in the `public/` folder.
